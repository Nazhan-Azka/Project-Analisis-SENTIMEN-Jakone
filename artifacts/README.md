# Artifacts

Folder ini berisi arsip/export hasil eksperimen yang dipindahkan dari proses Google Colab.

`hasil_indobert_v3_baseline/` adalah export lengkap eksperimen IndoBERT v3 baseline, termasuk dataset, lexicon, model, training log, dan evaluation.

Metrik utama:
- Validation accuracy: 0.919548
- Validation macro F1: 0.868058
- Test accuracy: 0.931594
- Test macro F1: 0.877363
- Test weighted F1: 0.930935
