# Evaluation IndoBERT V3 Baseline

Folder ini berisi hasil evaluation model IndoBERT v3 baseline pada data test.

Sumber hasil berasal dari eksperimen IndoBERT v3 baseline di Google Colab.

Metrik utama:
- Validation accuracy: 0.919548
- Validation macro F1: 0.868058
- Test accuracy: 0.931594
- Test macro F1: 0.877363
- Test weighted F1: 0.930935
