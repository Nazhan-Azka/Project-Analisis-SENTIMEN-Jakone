# Model IndoBERT V3 Baseline

Folder ini berisi model final hasil eksperimen IndoBERT v3 baseline dari Google Colab.

Metrik utama:
- Validation accuracy: 0.919548
- Validation macro F1: 0.868058
- Test accuracy: 0.931594
- Test macro F1: 0.877363
- Test weighted F1: 0.930935

Model ini digunakan sebagai model utama untuk evaluasi dan pelaporan skripsi.
