# Modeling IndoBERT V3 Baseline

Folder ini berisi hasil training dan validation dari eksperimen IndoBERT v3 baseline.

Sumber hasil berasal dari eksperimen fine-tuning IndoBERT v3 baseline di Google Colab.

Metrik utama:
- Validation accuracy: 0.919548
- Validation macro F1: 0.868058
- Test accuracy: 0.931594
- Test macro F1: 0.877363
- Test weighted F1: 0.930935
